### Key Design Decisions
1. **No Backend Integration Yet**: All API methods are placeholders. Backend integration will come in Step 4.
2. **Client-Side Auth Check**: Simple token check in dashboard layout. Will be enhanced in Step 4.
3. **WIP Placeholders**: All pages contain "WIP" indicators to show they're not yet implemented.
4. **Incremental Approach**: Following strict step-by-step process - UI first, backend later.
